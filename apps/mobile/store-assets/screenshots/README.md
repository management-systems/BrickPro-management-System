# Screenshots Guide (1080×1920 px — 9:16 ratio)

Take these 5 screenshots from your phone:

## 1. 01-dashboard.png
- Open app → Dashboard
- Shows: Monthly stats cards, quick action buttons
- Tip: Use with some data so it looks real

## 2. 02-production.png  
- Open app → Production
- Shows: Brick type summary cards + recent entries
- Tip: Add a few production entries first

## 3. 03-dispatch.png
- Open app → Sell Bricks
- Shows: Dispatch list with challan numbers, amounts, payment badges
- Tip: Show mix of PAID/PARTIAL/CREDIT entries

## 4. 04-reports.png
- Open app → Reports → Charts tab
- Shows: Revenue vs Expenses chart, profit trend
- Tip: Best with 2-3 months of data

## 5. 05-settings.png
- Open app → Settings
- Shows: Profile card, Premium plan pricing, Pay Now section
- Tip: Shows the subscription and UPI payment flow

---

## How to capture:
1. Open Expo Go app on your Android phone
2. Run `npx expo start` on your computer
3. Scan QR code with Expo Go
4. Navigate to each screen
5. Take screenshot (Power + Volume Down)
6. Transfer to this folder
7. Rename as shown above

## Size requirement:
- Exactly 1080×1920 pixels (9:16 portrait)
- If your phone has different resolution, resize using any photo editor
- Format: PNG or JPEG
